# E-Commerce Sales Analysis

## Project Description

This project analyzes e-commerce sales data to identify key business insights related to sales performance, profitability, products, categories, and regional performance.

The analysis was performed using Python, Pandas, Matplotlib, and Jupyter Notebook.

## Objectives

* Analyze sales and profit by category
* Analyze sales and profit by region
* Identify top-performing products
* Analyze sales trends over time
* Evaluate profitability across categories

## Key Findings

* Electronics generated the highest sales and profit.
* The West region achieved the highest sales and profit.
* Camera and Monitor were the best-performing products.
* Monthly sales fluctuated over time without a clear long-term trend.
* Profit margins were similar across all categories.
